The data pipeline flows as a sequence of operations from agent execution through evaluation to persistent storage and executive dashboarding.

The sequence begins when the deployed FinanceAgent receives queries through the Vertex AI Agent Engine runtime. As the agent processes each query, the Agent Engine automatically emits OpenTelemetry spans to Google Cloud Trace, capturing latency timings, token consumption counts, and invocation success or failure status for every reasoning step, LLM generation call, and tool execution.

In parallel, the orchestration script collects the agent's prompt-response pairs and sends them to the Vertex AI EvalTask service. The EvalTask applies a custom PointwiseMetric rubric that scores each response on a 1 to 5 scale across helpfulness and conciseness criteria, producing a numeric agent_quality_score and a written explanation for each evaluation.

The evaluation results are returned as a pandas DataFrame. The pipeline enriches this DataFrame by injecting two metadata columns: eval_timestamp recording the exact UTC time of the evaluation run, and agent_version tagging the current release identifier. Column names containing forward slash characters from the Vertex Eval output are sanitized by replacing slashes with underscores to comply with BigQuery naming requirements.

The enriched DataFrame is then written to BigQuery using the pandas-gbq library. The to_gbq function appends the data to the agent_metrics.eval_rubric_results table in append mode, preserving the full history of evaluation runs. The final BigQuery schema contains seven fields: eval_timestamp, agent_version, prompt, response, agent_quality_score, explanation, and exact_match.

Looker connects to this BigQuery table to provide executive dashboarding. Analysts can query average quality scores grouped by time period and agent version, track quality regressions across releases, and optionally correlate evaluation scores with operational latency data from Cloud Trace to identify whether performance degradation impacts response quality.
